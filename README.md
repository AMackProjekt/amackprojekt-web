# amackprojekt-web

A MackProjekt Website - A Division of MackEnterprises

## 🚀 Use This as a Template

**This repository is a production-ready website template system!** Create professional client websites in minutes, not days.

### Quick Start - 3 Ways to Use This Template:

#### Option 1: CLI Generator (⚡ Fastest - Recommended)

```bash
# Clone this repository
git clone https://github.com/AMackProjekt/amackprojekt-web.git
cd amackprojekt-web

# Install dependencies
npm install

# Run the generator
npm run create-client-site

# Follow the interactive prompts
# ✨ Your new site will be ready in ../your-client-name/
```

**Time to launch: 5-10 minutes!**

#### Option 2: GitHub Template Button

1. Click the **"Use this template"** button at the top of this page
2. Create your new repository
3. Clone and run `npm run create-client-site`

#### Option 3: Direct CLI with Flags

```bash
npm run create-client-site -- \
  --name "Acme Corporation" \
  --slug acme-corp \
  --domain acmecorp.com \
  --primary-color "#3B82F6" \
  --output ../acme-corp
```
