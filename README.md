## 🚀 Use This as a Template

**This repository is a production-ready website template system!** Create professional client websites in minutes, not days.

### Quick Start - 3 Ways to Use This Template:

#### Option 1: CLI Generator (⚡ Fastest - Recommended)

```bash
# Clone this repository
git clone https://github.com/AMackProjekt/amackprojekt-web.git
cd amackprojekt-web

# Install dependencies
npm install

# Run the generator
npm run create-client-site

# Follow the interactive prompts
# ✨ Your new site will be ready in ../your-client-name/
```

**Time to launch: 5-10 minutes!**

#### Option 2: GitHub Template Button

1. Click the **"Use this template"** button at the top of this page
2. Create your new repository
3. Clone and run `npm run create-client-site`

#### Option 3: Direct CLI with Flags

```bash
npm run create-client-site -- \
  --name "Acme Corporation" \
  --slug acme-corp \
  --domain acmecorp.com \
  --primary-color "#3B82F6" \
  --output ../acme-corp
```

## T.O.O.L.S Inc Portal Features

This project showcases a comprehensive portal system for **T.O.O.L.S Inc** (Together Overcoming Obstacles and Limitations), designed to support justice-involved individuals through programs, referrals, and resources.

### Portal Capabilities (Inspired by [Huly Platform](https://github.com/hcengineering/platform))

The portal includes 8 integrated features for managing participant programs and staff coordination:

1. **📋 Case Management** - Track and manage participant cases with detailed profiles, program enrollment, and progress monitoring
2. **✅ Task & Milestone Tracking** - Board and list views for managing participant assignments and program milestones
3. **📄 Document Management** - Secure document storage with categorization, search, and confidentiality controls
4. **💬 Team Collaboration** - Real-time messaging channels for staff coordination and participant communication
5. **📚 Knowledge Base** - Searchable wiki with articles, guides, and resources organized by category
6. **⏱️ Time Tracking** - Track program hours with timer, activity logging, and billable hours reporting
7. **📊 Activity Timeline** - Comprehensive activity feed with filtering and real-time updates
8. **🔔 Notifications** - Priority-based notification system with read/unread tracking

### Tech Stack

- **Framework**: Next.js 14.2.0 with App Router
- **UI**: React 18 + TypeScript + Tailwind CSS
- **Animation**: Framer Motion 11.0
- **Charts**: Recharts 2.12.7
- **Deployment**: Azure Static Web Apps
