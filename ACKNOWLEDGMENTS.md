# Acknowledgments & Credits

## Inspiration

This project's portal features were inspired by the excellent work of the **Huly Platform** team.

### Huly Platform
- **Repository**: [hcengineering/platform](https://github.com/hcengineering/platform)
- **Website**: [huly.io](https://huly.io)
- **License**: EPL-2.0

Huly is an all-in-one project management platform (alternative to Linear, Jira, Slack, Notion, Motion) that includes:
- Project Management with task tracking, kanban boards, and sprints
- CRM (Customer Relationship Management)
- Team collaboration and real-time chat
- Document management
- Time tracking
- Knowledge base/wiki

### What We Adapted

Our T.O.O.L.S Inc portal features were inspired by Huly's architecture and UI patterns, specifically:

1. **Case Management System** - Inspired by Huly's CRM module, adapted for tracking justice-involved individuals
2. **Task & Milestone Tracking** - Inspired by Huly's project management, adapted for participant program progress
3. **Document Management** - Inspired by Huly's document handling, designed for confidential participant files
4. **Team Collaboration** - Messaging and channels inspired by Huly's team features
5. **Knowledge Base** - Wiki-style resource center inspired by Huly's documentation system
6. **Time Tracking** - Program hours tracking inspired by Huly's time management
7. **Activity Timeline** - Real-time updates inspired by Huly's activity feeds

### Implementation Differences

While inspired by Huly, our implementation is:
- Built with **Next.js 14 + React** (Huly uses Svelte)
- Focused on **justice-involved support** use cases
- Designed for **non-profit organizations**
- Optimized for **Azure Static Web Apps** deployment
- Uses a **different tech stack** and architecture

## Other Technologies
